forceR: An R package to analyze bite forces

The R package forceR can be used to sort, clean, and analyze thousands of bite force measurements or other time series data. It includes amplifier drift corrections, an automatic and customizable bite finding algorithm to identify and extract the strongest peaks (bites) in a time series, and functions to fit polynomial models to the extracted bites for further analyses, e.g. a principal component analyses. The forceR package is available on CRAN and my GitHub page, where you can also find a detailed vignette. The original publication of the package is available in the same manuscript as the forceX setup in Methods in Ecology and Evolution.
